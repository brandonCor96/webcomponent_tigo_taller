## Run Locally

Clone the project

```bash
  git clone https://github.com/brandonCor96/webcomponent_tigo_taller.git
```

Go to the project directory

```bash
  cd webcomponent_tigo_taller
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run start
```

Visit src/wcTigoTaller/dist/index.html to See the card.
